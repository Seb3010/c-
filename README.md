# Proyecto C++: Hello World

Este repositorio es una bitaocra de aprendizajes, del lenguaje c++

## ¿Cómo compilar y ejecutar?

1. Abre una terminal en la carpeta del proyecto.
2. Compila el archivo usando el siguiente comando:

   ```sh
   g++ helloworld.cpp -o helloworld.exe
   ```

3. Ejecuta el programa:

   ```sh
   ./helloworld.exe
   ```

## Descripción

El archivo `helloworld.cpp` imprime un mensaje de "Hello, World!" en la consola. Es un ejemplo básico para comenzar a programar en C++.

## Requisitos
- Tener instalado un compilador de C++ (por ejemplo, [GCC](https://gcc.gnu.org/) o [MinGW](http://www.mingw.org/)).
